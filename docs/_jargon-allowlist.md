# Jargon allowlist

This file is **not** part of the published site. It's a working contract for the author: every AI-domain term that may appear in the notes is listed below. Any term you want to use that is **not** on this list must be glossed in plain language on first use, or removed.

The grep gate in the plan's verification steps backs this up, but the allowlist is the human checklist.

## Seed approved terms

- `LLM` — large language model
- `prompt` — the text you type in
- `context window` — how much text the model can "see" at once
- `hallucination` — confidently-stated wrong answer
- `chat model` — the kind of model you talk to in turns (e.g. ChatGPT, Claude)
- `agent` — an AI tool that can take steps on its own
- `RAG` — retrieval-augmented generation; giving the model documents to read before answering
- `embedding` — a numeric representation of text used for search/similarity
- `fine-tune` — extra training to specialize a model
- `token` — the model's unit of text (roughly a word fragment)

## Rule

If you write a note and want to use a term not on this list, add it here with a one-line gloss, or rephrase the sentence to avoid it.
